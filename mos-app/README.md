# mos-app

The Gordi MOS web app (React 19 + Vite + TypeScript), served at `https://ops.gordi.id/mos`.

Project context, operating model, and quality gates live in the repo-root [`CLAUDE.md`](../CLAUDE.md).
Run all commands from this directory: `npm run dev` · `npm run typecheck` · `npm run lint:ci` · `npm test` · `npm run build` · `npm run e2e`.
